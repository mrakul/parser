# Lesson 3.2.2 — Serde API Practice

Решение практической задачи: структуры для JSON API интернет‑магазина с использованием Serde.

## Ключевые особенности

- `Product` сериализует цену в копейках, скрывает `internal_id`.
- `Order` прячет внутренний идентификатор, валидирует email и держит `created_at` только для десериализации.
- `OrderStatus` сериализуется в виде строки (`"pending"`, `"processing"` и т. д.).
- Unit-тесты проверяют сериализацию/десериализацию и валидацию.

## Запуск тестов

```bash
cargo test
```
